# 支持少量BBCode修改输出文本的样式

`[color=十六进制颜色或颜色名称]文本[/color]`或`[c=十六进制颜色或颜色名称]文本[/c]`：将文本设置为指定的颜色。
`[bold]文本[/bold]`或`[b]文本[/b]`：将文本设置为加粗。
`[italic]文本[/italic]`或`[i]文本[/i]`：将文本设置为斜体。
`[underline]文本[/underline]`或`[u]文本[/u]`：将文本设置为下划线。
`[strikethrough]文本[/strikethrough]`或`[s]文本[/s]`：将文本设置为删除线。
`[background=十六进制颜色或颜色名称]文本[/background]`或`[bg=十六进制颜色或颜色名称]文本[/bg]`：将文本设置为背景颜色。
`[url=链接地址]文本[/url]`：将文本设置为链接。

# 特殊用法

`[color=#十六进制颜色 #十六进制颜色]文本[/color]`或`[c=#十六进制颜色 #十六进制颜色]文本[/c]`：将文本设置为指定两个颜色的从左至右的渐变，从第一个颜色开始，到第二个颜色结束；颜色填写越多，渐变的色彩点越多，但是都作为长度的等分点的一个颜色锚点，最后一个颜色锚点为文本的结束位置。
`[background=#十六进制颜色  #十六进制颜色]文本[/background]`或`[bg=#十六进制颜色 #十六进制颜色]文本[/bg]`：将文本背景设置为指定多个颜色的从左至右的渐变，从第一个颜色开始，到最后一个颜色结束；颜色填写越多，渐变的色彩点越多，但是都作为长度的等分点的一个颜色锚点，最后一个颜色锚点为文本的结束位置。
`[bold=粗细]文本[/bold]`或`[b=粗细]文本[/b]`：将文本设置为指定的粗细。
`[underline=粗细]文本[/underline]`或`[u=粗细]文本[/u]`：将文本设置为指定的下划线粗细。
`[strikethrough=粗细]文本[/strikethrough]`或`[s=粗细]文本[/s]`：将文本设置为指定的删除线粗细。

# 转义

使用`\`可以转义字符以显示原字符不被解析为标签等，比如`\[`, `\]`, `\\`等。
